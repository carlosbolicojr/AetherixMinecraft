## 📝 Sobre o Projeto

O **Portal Aetherix** foi desenvolvido para centralizar a experiência de jogadores de Minecraft. O foco principal foi a criação de uma interface limpa, com padronização de navegação entre diferentes páginas e uma identidade visual forte (Dark Theme).

### 🛠️ O que foi implementado?

- **Header Padronizado:** Navegação fixa utilizando `position: fixed` para garantir que o menu acompanhe o scroll do usuário em todas as páginas.
- **Loja Interativa:** Tabelas comparativas de planos (VIPs) com efeitos de `hover` e escala via CSS.
- **Suporte ao Usuário:** Formulário de contato estruturado para recebimento de solicitações da Staff.
- **Área Jurídica:** Páginas dedicadas a Termos de Uso e Regras, garantindo a transparência com o jogador.

## 🚀 Como visualizar

1. Faça o download ou clone este repositório.
2. Certifique-se de que a pasta `css` e `imagem` estão no mesmo nível dos arquivos HTML (ou conforme a estrutura do repo).
3. Abra o arquivo `inicio.html` no seu navegador favorito.
